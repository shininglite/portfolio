About Me

I am a recent graduate of an intensive full-stack certificate web development program at University of Minnesota College of Continuing and Professional Studies. There, I learned modern technologies that are widely used to develop and deploy responsive, interactive websites. These include MongoDB database, and Javascript libraries: Express JS, React JS, and Node JS. Varied projects require knowledge of jQuery syntax, JSON data format, AJAX API calls, GitHub and Heroku deployment.

As I have now completed this certificate, I'm pleased with the breadth of tools and technologies to which I have been introduced and had a chance to explore. In particular, I enjoyed building a full-stack game with a project partner. I am continuing to improve my coding skills through online tutorials, reading, research, and networking. I am also designing and programming my own full-stack Mongo, Express, React, Node (MERN) projects.
